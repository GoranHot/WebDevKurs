Za GIT

- Crvena boja - kad se brise fajl sa GIT-a
- Zelena boja - kad se dodaje fajl na GIT
- Zuta boja - kad se mijenja fajl

(dodati u dati direktorij)
git clone "link"
primjer
git clone https://github.com/GoranHot/WebDevKurs.git


za dodavanje/selektovanje
(specificno)
git add .\readme2.txt

(dodavanje/selektovanje svega izmijenjenog, ## space izmedju add i tacke ##)
git add .

(commit - lokalno)
git commit -m "prvi commit"

(push da ide online)
git push
